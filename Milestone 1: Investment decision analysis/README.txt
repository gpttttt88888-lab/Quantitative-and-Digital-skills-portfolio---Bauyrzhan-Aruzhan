Portfolio Analysis – Adidas & Microsoft


## Overview
Analysis of historical stock data for Adidas AG (ADS.DE) and Microsoft Corporation (MSFT) from March 2025 to March 2026. Demonstrates quantitative and digital skills through calculation of returns, volatility, and visual trends.


## Data & Methods
- Source: Yahoo Finance
- Calculated daily and weekly returns
- Generated charts for price trends and weekly returns
- Key metrics: mean, range, standard deviation, IQR
- Outliers identified using IQR method


## Skills Demonstrated
- Data collection and cleaning
- Excel calculations and statistical analysis
- Graphical visualization of financial data
- Summarizing insights in a professional format
